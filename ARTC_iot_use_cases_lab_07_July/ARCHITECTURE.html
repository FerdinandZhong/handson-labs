UC1
Predictive Maintenance — Remaining Useful Life (RUL)
Random Forest Regressor · predicts hours until machine failure
Data Sources
Raw Sensor Logs
Per-machine 5-min snapshots of vibration (X/Y/Z axes) and health index
machine_health.csv rul_predictions.csv
Merge & Align
Temporal Join
merge_asof per machine (±10 min tolerance) · interpolate missing RUL labels
Feature Eng.
Derived Features
avg_rms, max_rms, z_x_ratio (axis imbalance), health_slope (rolling 3-window degradation rate)
Training
Random Forest
100 trees · 80/20 split · evaluated on MAE + RMSE
rul_model.pkl feature_list.json
Serving
FastAPI App
POST /predict · GET /health · confidence from RF tree spread
app.py run_app.py
Input Features (9 total)
FeatureUnitDescription

UC2
Predictive Quality — Defect Rate & Risk Level
XGBoost Regressor + Classifier · dual-model: defect rate (%) + risk category
Data Sources
Sensor Streams
Narrow-format per-metric sensor readings + quality label predictions
sensor_readings.csv quality_predictions.csv
Pivot & Merge
Wide Format Join
pivot_table narrow→wide per (machine_id, event_time) · missing metrics filled 0.0 · merge_asof ±5 min
Training
Dual XGBoost
XGBRegressor → defect_rate · XGBClassifier → risk_level (4-class) · 80/20 split
quality_regressor.pkl quality_classifier.pkl risk_encoder.json feature_list.json
Serving
FastAPI App
POST /predict · confidence from classifier.predict_proba() max probability
app.py run_app.py
Input Features (6 total — all optional, default 0.0)
FeatureUnitProcess Types
vibration_rmsmm/sall machines
temperature°Call machines
sound_dbdBcnc, paint_booth
humidity%paint_booth, battery_assembly
pressurekPacnc, battery_assembly
voltageVbattery_assembly
process_type & machine_id are metadata — echoed in response, not used as model features.
Algorithm & Output
Regressor XGBRegressor · 100 estimators · predicts defect_rate (clamped 0–1)
eval_metric = rmse
Classifier XGBClassifier · 100 estimators · predicts risk_level (4-class ordinal)
Confidence = max(predict_proba) · fallback 0.80
defect_ratefloat 0–1Predicted proportion of defective units
risk_levelstringQuality risk category
confidencefloat 0–1Classifier max class probability
feature_windowobjectEcho of key inputs for traceability
LOWcode 0
MEDIUMcode 1
HIGHcode 2
INVESTIGATEcode 3
